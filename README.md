# NewsClientApp 📱



Учебное мобильное приложение на базе Android для просмотра свежих новостей из мира космонавтики и науки.

## Функционал приложения

* **Загрузка данных из сети:** Получение списка статей с помощью REST API.

* **Архитектура:** Разделение бизнес-логики и UI, что позволяет приложению сохранять состояние/данные при повороте экрана.

* **Пагинация:** Динамическая подгрузка новых статей (по 25 элементов) при доскролле списка до самого низа карточек.

* **Асинхронность:** Сетевые запросы выполняются в фоновом потоке с использованием `Kotlin Coroutines`.

* **Обработка ошибок:** Корректная обработка сбоев сети и ссылок на картинки без падения приложения.
